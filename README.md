# 💄 Teju Beauty - E-Commerce Storefront

A sleek, responsive, and modern web application built to provide a seamless online shopping experience for exclusive beauty and skincare products.

## 🚀 Features
* **Modern Navigation:** Clean header navigation including Home, Products, About Us, and Contact sections.
* **User Authentication:** Built-in interfaces for User Login and Registration (Sign Up) pages.
* **Product Showcase:** A dedicated "Featured Products" section to highlight top-tier inventory.
* **Responsive Layout:** Optimized for a smooth user experience across various device screens.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3
